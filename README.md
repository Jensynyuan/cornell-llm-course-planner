# LL.M. Course Planner v5.21

本机离线课程规划工具，默认提供 Cornell Fall 2026 LAW 数据，并支持其他法学院按“官方材料 → AI 整理 → ZIP 导入”的方式扩展。

## v5.21 重点

- 修复 NY Bar 24 学分核心计算：不再只依赖少量预标注课程，而是按用户实际选择的班次判断。康奈尔法学院线下讲授、研讨和讨论班次会自动计入；线上、独立研究和官方明确排除的班次不计入；诊所、实践课及项目限制班次会明确提示“需院系确认”。
- 课程筛选、课程卡、已选课程摘要和学分进度统一改用“NY Bar 课堂学分／需院系确认”表述，避免把一般线下法学院课程误显示为不计入 24 学分的“外选修”。
- 分类学分同样按所选班次计入，防止线上或明确排除的班次被误算入职业责任、法律研究与写作、美国法体系或 NYLE / Bar 考试科目。

## v5.20 重点

- 增加面向网页访问者的本地保存说明：明确正常刷新不会清空课表，不同用户不会共享数据，并提示换设备、无痕窗口和清除网站数据的边界。
- “我的课表”新增 JSON 本地备份与恢复，可在更换浏览器、设备或网站域名前主动迁移个人课表、课程编辑、外校导入和偏好。
- 增加 Cloudflare Pages 生产部署配置和两个无状态 Pages Functions；不绑定 KV、D1、R2 或其他共享数据库。
- 远程数据读取增加内网地址、凭据重定向、超时与体积防护；ZIP 导入增加路径、加密、条目数和解压体积校验。

## v5.19 重点

- 搜索会同时匹配 NY Bar 分类名称；搜索“法律研究与写作”会返回本学期已开设、并标注为该分类的 LAW 6761。
- 课表中不足 70 分钟的课程改为紧凑双行布局，优先保留课程代码、时间、教师和授课地点；冲突课程也会保留这些字段。
- 本学期 LLM 注册学分超过公布上限时，进度条和提示框会变为红色，并直接突出超出学分和“需申请批准”。

## v5.18 重点

- 周课表的重叠班次改为均分左右栏显示，外侧以单一红框标出整个重叠区，避免课程卡相互遮挡。

## 启动

双击 `start.bat`，浏览器将打开 `http://127.0.0.1:4185`。

## 导入其他学校

打开“学校与数据”，选中学校后：

1. 点击官方课程源并保存目标学期的完整页面、导出文件或网页源代码；
2. 下载本校操作清单，复制页面中的 AI 导入提示词；
3. 将材料交给可使用的 AI，让其生成无密码 ZIP；
4. 上传 ZIP、读取预览、查看导入报告，再保存并自动切换学校。

完整说明见 `T14_IMPORT_GUIDE.md`；字段结构见 `IMPORT_SCHEMA.md`。

## 数据限制

数据只保存在当前设备浏览器。软件不会绕过登录、权限或访问控制。NY Bar、学位及签证资格必须以学校和主管机构的最终确认为准。
